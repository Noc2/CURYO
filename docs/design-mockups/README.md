# Cleaner Design Mock-ups

These are two standalone visual concepts for Curyo, created as review artifacts rather than production UI.

## Concepts

1. `planet-flare-signal.svg`
   Inspired by the Spacefox reference.
   Large cinematic signal planet, dark interface shell, and a bright orbital flare that can map to rating activity or live reputation flow.
   Preview export: `planet-flare-signal.png`

2. `soft-orb-discover.svg`
   Inspired by the Better Problems reference.
   Softer ambient gradients, a larger color-washed orb, and a calmer Discover surface with cleaner signal chips and rating modules.
   Preview export: `soft-orb-discover.png`

## Notes

- Both concepts keep the current "signal planet" idea, but push it further as the main visual anchor.
- These are mock-ups only.
- The live app was reverted separately and is not using these designs.
